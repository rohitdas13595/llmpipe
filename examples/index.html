llmpipe examples

Run the matching server from the llmpipe/ module root:

  • go run ./cmd/voicebot → WebSocket voice client (mic → PCM → /ws).
  • go run ./cmd/voicebot-livekit → LiveKit voice client (joins room, token from /api/livekit-token; default demo HTTP :8090).
  • go run ./cmd/example-minimal-pipeline — no network: one processor + observer, prints a TextFrame (template for custom stages).
